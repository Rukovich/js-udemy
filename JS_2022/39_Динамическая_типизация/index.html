Динамическая типизация

Динамическая типизация- возможность одного типа данных превращаться в другой

Преобразовываем в строку: to String

1 Способ: typeof

console.log(typeof(String(null))); //string
console.log(typeof(String(4))); //string

2 Способ: конкатинация

console.log(typeof(5 + '')); //string

Пример:

const num = 5;
console.log('https://vk.com/catalog/' + num);

Преобразовываем в число: to mumber

1 Способ: typeof

console.log(typeof(Number('4'))); //number

1 Способ: унарный +

console.log(typeof(+'5')); //number

3 Способ: parseInt

console.log(typeof(parseInt('15px', 10))) //number

Преобразовываем в булиновое значение: to boolean

Что является false: 0, '', null, undefined, NaN;

все остальное - true

1 Способ:

let switcher = null;
if (switcher) {
    console.log('working...'); // не работает
}

switcher = 1;
if (switcher) {
    console.log('working...'); // работает
}

2 Способ:

console.log(typeof(Boolean('4'))); //Boolean

3 Способ:

console.log(typeof(!!'444')); //Boolean
