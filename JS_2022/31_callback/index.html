Callback-функции

callback- функция котора должна быть выполнена после того, как другая функция завершила свое выполнение

Многие действия в JavaScript асинхронные. Для того, чтобы запускать функции в нужной последовательности можно использовать Callback-функции

function learnJS(lang, callback) {
  console.log(`Я учу ${lang}`);
  callback();
}
function done() {
  console.log('Я прошел этот урок');
}

learnJS('JavaScript', done);

Код выше сначала выведет:
Я учу JavaScript,
затем:
Я прошел этот урок
