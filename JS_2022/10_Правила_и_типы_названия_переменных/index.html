Правила и типы названия переменных

Как не стоит называть переменные

      
        const a = 5000;
        const b = 4000;
        console.log('Ширина кузова автомобиля:' + a + ', длина' + b);

Как стоит

    
      const vehicleBodyWidth = 5000;
      const vehicleBodylenght = 4000;
      console.log('Ширина кузова автомобиля:' + vehicleBodyWidth + ', длина' + vehicleBodylenght);

Какие бывают стили написания:

  • snake_case
  • UPPER_SNAKE_CASE- обычно так обозначают константы (COLOR_RED = '#F00');
  • _apiKey- такое тоже лучше не трогать;
  • kebab-case- слова через "-";
  • PascalCase- Первая буква большая;
