Методы и свойства

Все значения в JavaScript, за исключением null и undefined, содержат набор вспомогательных функций и значений, доступных «через точку».

Такие функции называют «методами», а значения – «свойствами». Здесь мы рассмотрим основы использования свойств и методов.

свойства

свойства -это вспомогательные значения. Свойства указываются через точку после выражения

Все свойства строки можно посмотреть с помощью команды в консоли console.dir('')

свойство lenght

Указывает количество символов в строке

const str = 'test';
console.log(str.length);

Методы

Методы- это вспомогательные функции.

toUpperCase

возвращает значение строки, преобразованное в верхний регистр

const str = 'tEsT
console.log(str.toUpperCase()); //TEST

toLowerCase

возвращает значение строки, преобразованное в нижний регистр

const str = 'TEST';
console.log(str.toLowerCase()); //test

indexOf()

возвращает индекс первого указанного значения на котором он был вызван. В примере ниже искомое слово 'fruit' начинается с 5 позиции.

const fruit = 'some fruit';
console.log(fruit.indexOf('fruit')); //5

Number.isInteger()

определяет, является ли переданное значение целым числом.

trim()

даляет пробельные символы с начала и конца строки

let orig = '   foo  ';
console.log(orig.trim()); // 'foo'

Методы позволяющие Взаимодействовать со строками

slice()

извлекает часть строки и возвращает новую строку без изменения оригинальной строки

const logg = 'hello world';
console.log(logg.slice(6, 11)); //world

Если задать только первый аргумент, метод вырежет всю строку начиная с укаханного индекса

const logg = 'hello world';
console.log(logg.slice(6)); //world (полностью вырезает строку начиная с 6 символа)

substr()

работает так же как slice или substring, только указывает длинну символов, которые нужно вырезать

const logg = 'hello world';
console.log(logg.substr(6, 5)); //world

Методы позволяющие Взаимодействовать c числами

Math.round

Округляет указанное число до ближайшего целого.

const num = 12.2;
console.log(Math.round(num));

parseInt()

принимает строку в качестве аргумента и возвращает целое число в соответствии с указанным основанием системы счисления

const test = '12.2px';
console.log(parseInt(test)); //12

parseFloat()

принимает строку в качестве аргумента и возвращает десятичное число (число с плавающей точкой)

const test = '12.2px';
console.log(parseFloat(test)); //12.2
