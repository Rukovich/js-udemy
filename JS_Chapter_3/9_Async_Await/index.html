Async / Awayt

Async / Awayt предназначены для работы с промисами и работают только парно.
async - говорит, что внутри функции будет какой-то асинхронный код
awayt - ставим перед теми операциями, корторые необходимо дождаться

Пример из проекта к этому уроку

  const postData = async (url, data) => {
  const res = await fetch(url, {
    headers: {
      "Content-type": "application/json",
    },
    method: "POST",
    body: data,
  });

  return await res.json(); //Обрабатываем как JSON формат
};

В этом уроке мы переписали проект так, чтобы данные для карточки поступали из сервера
