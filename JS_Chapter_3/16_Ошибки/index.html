Чтобы ошибки не сломали весь код, можно использовать конструкцию try / catch

Конструкция try...catch пытается выполнить инструкции в блоке try, и, в случае ошибки, выполняет блок catch.

Синтаксис:

try {
    console.log('normal');
    console.log(a); //ошибка
    console.log('result'); 
}  catch(error) { //объект ошибки
    console.log(error.name); //название ошибки
    console.log(error.message); //сообщение
    console.log(error.stack); //стек ошибок

    //Не обязательный блок- завершает операции в любом случае
}  finally { 

}

Еще пример

try {
  myroutine(); // может выбрасывать три вида исключений
} catch (e) {
  if (e instanceof TypeError) {
    // обработка исключения TypeError
  } else if (e instanceof RangeError) {
    // обработка исключения RangeError
  } else if (e instanceof EvalError) {
    // обработка исключения EvalError
  } else {
    // обработка остальных исключений
    logMyErrors(e); // передать обработчику ошибок
  }
}
